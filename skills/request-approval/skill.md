# request-approval

Create a human-in-the-loop approval request file for sensitive actions.

## Trigger

Use this skill when about to take ANY of these actions:
- Send an email to a new/unknown contact
- Send any email (reply or new)
- Make or schedule any payment
- Post on social media (LinkedIn, Twitter, Instagram, Facebook)
- Delete any file
- Contact any external party on behalf of the business
- Any action involving money > $50

NEVER act directly on these — always create an approval file first.
