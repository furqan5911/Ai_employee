# generate-briefing

Generate the Monday Morning CEO Briefing - proactive weekly business summary.

## Trigger

Use this skill when:
- It's Monday morning (scheduled at 7:00 AM)
- User says: "generate briefing", "CEO briefing", "weekly summary", "business audit"
- User asks: "how's business", "what happened this week", "weekly report"

## When NOT to Use

- DO NOT use automatically on every Monday (wait for user or scheduled task)
- DO NOT use if briefing already exists for this week
- DO NOT use mid-week (unless user requests)

## Instructions

### Silver Tier Mode (Current)

Since full accounting data isn't available yet (Gold tier feature):

1. **Determine the period**
   - Default: last 7 days (Monday to Sunday)
   - Use today's date for filename

2. **Gather available data**
   - Read `Business_Goals.md`: revenue targets, active projects
   - Scan `/Done/` folder: count completed items this week
   - Scan `/Needs_Action/`: identify pending items
   - Scan `/Plans/`: identify active/incomplete plans

3. **Generate insights**
   - What was accomplished this week?
   - What's still pending?
   - Any bottlenecks or delays?
   - Upcoming deadlines (next 14 days)

4. **Write the briefing**
   - File: `/Briefings/YYYY-MM-DD_Monday_Briefing.md`
   - Keep it executive-level: concise, actionable
   - Note if data is limited (Silver tier)

### Gold/Platinum Tier Mode (Future)

When accounting is connected:
- Read `Accounting/Current_Month.md`: income, expenses
- Analyze subscriptions and recurring costs
- Calculate actual revenue vs targets
- Generate proactive cost-saving suggestions
